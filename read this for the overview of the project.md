"Brainstormer Quiz" (Manual Testing Project)

Tester: Vighnesh Hegde

Project Overview-
A simple quiz-based web application containing 4 pages consisting of 3 questions with 2 options each for answer and 1 login page created intentionally with bugs to practice manual testing;).

Application Flow-
Login >> Question1 >> Question2 >> Question3

Testing Concepts Used:

                      -SDLC & STLC
                      -Smoke Testing
                      -Functional Testing
                      -Black Box Testing
                      -Negative Testing
                      -Positive Testing
                      -Exploritory Testing

Intentional Bugs:

                 1.Login button allows users to log in  below 13 years and above 100 years
                 2.Login button allows users to log in with no credentials
                 3.Wrong answer in Question 1 still navigates forward
                 4.Quit Game button leads to error page
                 5.Question 2 wrong answer has no action
                 6.Question 3 wrong answer leads to dead-end page



Will be delighted to add new bug reports, so do comment if you find one ;).
